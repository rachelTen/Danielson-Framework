# Danielson-Framework

## Product Summary
The Charlotte Danielson Framework, is an extensive non quanitifiabe tool use to either evaluate teachers and for teachers to evaluate themselves. Due to the length, the problem exists - where does a teaching professional find the time to do the assessment. This project automizes the process, providing quick reference guides, and quantifiable figures a teacher can use to enhance their practice. 

## Target Users
Teaching professionals, adminstrators in education

## Current Stage
Teacher Evaluation App  beta phase active development working on stability uses, and auto-saving feature. 

## Built With
Bolt

## Core Features
Core detailed assessment full
Shorten version of assessment
Ability for user to download assessment for review

## Product Logic
Teacher creates a profile, the landing page allows for the teacher to name their evaluation, if past evaluations were done it shows visibly on the the landing page. The Danielson framework has four domains. When the teacher/user creates a new evaluation it presents them with the first domain, and other pages follow until the fourth and final domain is reached. The ability to save happens on the fourth domain page.
